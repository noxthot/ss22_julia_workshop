Loops

Just as any scientific programming language, Julia has different variants to implement loops. The perhaps most common one is the for loop

for i in 1:N
    # loop body
end

Here, Julia performs instructions inside the loop body for i=1,⋯ ,Ni = 1,\cdots,Ni=1,⋯,N. You can modify the iterates by for i in indexStart:increment:indexEnd. Hence, if we want to iterate from NNN to 111 with an increment of 222, we have 

for i in N:-2:1
    # loop body
end

If we want to iterate over a list or vector vvv, we can use for i in v. In this case, iii will take values in the vector vvv.

A further loop that can be used is the while loop, which performs a given instruction as long as a specified condition holds. Our previous example written as a while loop looks like this

while i <= N
    # loop body
    i = i - 2
end
As an exercise, we compute the dominant eigenvalue of a matrix AAA. That is, we wish to find the maximal scalar values λ\lambdaλ such that there exists a vector vvv which fulfills Av=λvAv = \lambda vAv=λv.

  1. Create a matrix A∈R5×5A\in\mathbb{R}^{5 \times 5}A∈R5×5 with entries Aij=(i−j)2A_{ij} = (i-j)^2Aij​=(i−j)2 using a for loop.

  2. Apply this matrix to a random vector v = rand(5) and normalize the result. That is, vnew=Av/∥Av∥v_{\mathrm{new}} = Av/\Vert Av\Vertvnew​=Av/∥Av∥.

  3. Repeat this process using vnewv_{\mathrm{new}}vnew​ as input (i.e., v←vnewv\leftarrow v_{\mathrm{new}}v←vnew​) until ∥vnew−v∥<10−5\Vert v_{\mathrm{new}}- v \Vert < 10^{-5}∥vnew​−v∥<10−5 using a while loop.

  4. Print out A⋅vnewA\cdot v_{\mathrm{new}}A⋅vnew​ and check against the eigenvalues of AAA using eigvals(A).

Solution 

using LinearAlgebra
# 1.
N = 5
A = zeros(N,N)

for i in 1:N
    for j in 1:N
        A[i, j] = (i - j) ^ 2;
    end
end

# 2.
v = rand(N);
vNew = A * v / norm(A * v)

#3.
while norm(v - vNew) > 1e-5
    v .= vNew
    vNew .= A * v ./ norm(A * v)
end
#4.
println("Approximated dominant eigenvalue is ", norm(A * vNew))

In order to stop a loop or skip an evaluation of the loop body at a specific iteration index if a certain condition is fulfilled we can use the break and continue commands. The break command will exit the loop. A simple example is

for i = 1:10
    if i > 5
        break
    end

    println("Iteration index is ", i)
end
println("Loop stopped.")

This will print out the iteration index untill a value of i=6i=6i=6 is reached at which point the loop will be exited. The continue statement allows us to skip an iterate.

for i = 1:10    
    if i == 5
        continue
    end

    println("Iteration index is ", i)
end

This prints out numbers from 111 to 101010 but skips the number 555.

Note that opposed to the if statement, loops are not leaky. That is, new variables defined inside a loop will be unknown outside the loop body.
I would like to reuse the result of this example/exercise in the parallel section
For a vector V∈RnV\in\mathrm{R}^nV∈Rn with elements vvv compute the sum of all the elements in the following fashion

  1. Sum over the elements per index:

s1=∑i=1nvis_1 = \sum_{i=1}^n v_is1​=i=1∑n​vi​

  1. Sum over the elements (hint for each):

s2=∑v∈Vvs_2 = \sum_{v\in V} vs2​=v∈V∑​v

  1. Transform these loops into functions

  2. Test against V = rand(100_000) and s_1 ≈ s_2 (use \approx + TAB for ≈)

Extra: Check the @inbounds macro for the first version.

Solution 

function mysum1(V)
    s = zero(eltype(V))
    for i in eachindex(V)
        s += V[i]
    end
    return s
end

function mysum2(V)
    s = zero(eltype(V))
    for v in V
        s += v
    end
    return s
end

V = rand(100_000)
mysum1(V) ≈ mysum2(V)
isapprox(mysum1(V),  mysum2(V); atol=1e-10, rtol=1e-10)
